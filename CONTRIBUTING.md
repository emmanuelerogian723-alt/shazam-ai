# Contributing to SHAZAM AI

Thank you for wanting to contribute! SHAZAM AI is 100% community-driven.

## How to Contribute

### Report a Bug
Open an issue at https://github.com/emmanuelerogian723-alt/shazam-ai/issues
Include: what happened, what you expected, your OS and Python version.

### Suggest a Feature
Open an issue with the label `enhancement`.
Describe the feature and why it would help others.

### Submit Code

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-feature`
3. Write your code with tests
4. Commit: `git commit -m "feat: describe your change"`
5. Push: `git push origin feature/your-feature`
6. Open a Pull Request

### Code Style
- Python: follow PEP8, use type hints, write docstrings
- All functions should be async where possible
- Add logging with structlog
- Handle errors gracefully — never crash silently
